Onboarding Flow

Registration & Approval

To ensure the safety and authenticity of the platform, all users must undergo a strict verification process before accessing the system.

1

User Registration

Hotel / NGO / Volunteer

User submits details (Name, Address, License/ID) via the signup form.

Status: Pending
2

Admin Review

Verification Process

Admin receives notification and manually validates submitted documents and profile authenticity.

Manual Check
3

Account Approval

Status Update

Upon verification, Admin approves the account. System sends an approval email to the user.

Approved
4

Access Granted

Login Enabled

User can now log in to their specific Role Dashboard and start using platform features.

System Live
